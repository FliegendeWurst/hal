◆ from_string()

BooleanFunction hal::BooleanFunction::from_string ( std::string  expression,
const std::vector< std::string > &  variable_names = {} 
)
static

Parse a function from a string representation. Supported operators are NOT ("!", "'"), AND ("&", "*", " "), OR ("|", "+"), XOR ("^") and brackets ("(", ")"). Operator precedence is ! > & > ^ > |.

Since, for example, '(' is interpreted as a new term, but might also be an intended part of a variable, a vector of known variable names can be supplied, which are extracted before parsing.

If there is an error during bracket matching, X is returned for that part.

Parameters
[in]expression- String containing a Boolean function.
[in]variable_names- List of variable names.
Returns
The Boolean function extracted from the string.

std::vector< BooleanFunction::Value > hal::BooleanFunction::get_truth_table ( std::vector< std::string >  ordered_variables = {},
bool  remove_unknown_variables = false 
) const

Get the truth table outputs of the function.

WARNING: Exponential runtime in the number of variables!

Output is the vector of output values when walking the truth table from the least significant bit to the most significant one.

If ordered_variables is empty, all included variables are used and ordered alphabetically.

Parameters
[in]ordered_variables- Variables in the order of the inputs.
[in]remove_unknown_variables- If true, all given variables that are not found in the function are removed from the truth table.
Returns
The vector of output values.
