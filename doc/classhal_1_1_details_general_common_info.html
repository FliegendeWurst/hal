Detailed Description

template<typename T>
class hal::DetailsGeneralCommonInfo< T >

Helper wrapper class.

A convenience class used to gather and extract the commonly shared information (such as name, type, id) of a module, gate and net in a single place. It functions as sort of a wrapper class.
