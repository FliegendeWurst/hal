class Value¶

Represents the logic value that a boolean function operates on.

Members:

ZERO : Represents a logical 0.

ONE : Represents a logical 1.

Z : Represents a high-impedance value.

X : Represents an undefined value.

static from_string(expression: str, variable_names: List[str] = []) → hal_py.BooleanFunction¶

Parse a function from a string representation. Supported operators are NOT (!, '), AND (&, *, ␣), OR (|, +), XOR (^) and brackets ((, )). Operator precedence is ! > & > ^ > |.

Since, for example, ( is interpreted as a new term, but might also be an intended part of a variable, a vector of known variable names can be supplied, which are extracted before parsing.

If there is an error during bracket matching, X is returned for that part.

Parameters:
  • expression (str) – String containing a Boolean function.
  • variable_names (list[str]) – List of variable names.
Returns:

The Boolean function extracted from the string.

get_truth_table(self: hal_py.BooleanFunction, ordered_variables: List[str] = [], remove_unknown_variables: bool = False) → List[hal_py.BooleanFunction.Value]¶

Get the truth table outputs of the Boolean function.

WARNING: Exponential runtime in the number of variables!

Output is the vector of output values when walking the truth table from the least significant bit to the most significant one.

If ordered_variables is empty, all included variables are used and ordered alphabetically.

Parameters:
  • ordered_variables (list[str]) – Variables in the order of the inputs.
  • remove_unknown_variables (bool) – If true, all given variables that are not found in the function are removed from the truth table.
Returns:

The list of output values.
Return type:

list[hal_py.BooleanFunction.Value]
